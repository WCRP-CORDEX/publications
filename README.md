# CORDEX publications and science monitoring

This repo collects discussions and resources regarding CORDEX publications and CORDEX science monitoring in general.
CORDEX publications are curated by the Points of Contact (POC) and coordinators of CORDEX activities (CORDEX-Domain, CORDEX-CORE, Flagship Pilot Studies, ...) using this Zotero library:

 * https://www.zotero.org/groups/5816477/wcrp-cordex/library

This repository also collects tools to exploit the Zotero library using its API to tailor publication lists for different purposes.

Science monitoring also includes research projects, which are collected via [this form](https://forms.gle/GSDaaDLPtaHkT5oq7). If you have a funded project supporting significantly any of the CORDEX activities, please fill out the form with the details.
